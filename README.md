# 📝 Text Summarizer Chatbot using Hugging Face T5-Small

An AI-powered Text Summarizer built using **FastAPI**, **Hugging Face Transformers**, and the **T5-Small** model. The application provides a simple web interface where users can enter long text and receive concise, meaningful summaries.

---

## 🚀 Features

- 📄 Automatic text summarization
- 🤖 Powered by Hugging Face T5-Small Transformer
- ⚡ FastAPI backend with REST API
- 🌐 Interactive web interface using HTML, CSS, and JavaScript
- 🔥 Uvicorn server for fast deployment
- 📱 Responsive and user-friendly UI
- 🎯 Supports long paragraph summarization

---
